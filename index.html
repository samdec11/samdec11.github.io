Meet Me Halfway
Code
Site

This was a group project designed to make planning outings with friends easier. It allows someone to enter up to ten different addresses, uses the Google Maps API to find the geographic center of those addresses, and then allows the person to run a search using the Yelp API to find the top five results around that center.
